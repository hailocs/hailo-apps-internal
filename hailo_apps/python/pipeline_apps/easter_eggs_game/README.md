# Easter Eggs Game

An interactive Easter egg catching game using Hailo pose estimation.

## How It Works

- A custom background image is displayed
- An Easter egg is placed at a random position on the screen
- Use your hands (wrists) to catch the egg
- When caught, the egg respawns at a new random location and your score increases
- Game runs for 60 seconds

## Usage

```bash
python easter_eggs_game.py --input usb --background /path/to/background.png
```

## Controls

- Move your hands to catch eggs
- Score is displayed on screen
- Game ends after 60 seconds
